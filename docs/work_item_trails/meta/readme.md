# Work Item Trails

Данный каталог содержит правила сопровождения Work Items.

Work Item рассматривается как самостоятельная единица дальнейшей работы.

Основной целью Work Item является не долговременное хранение знаний, а сохранение исследовательских вопросов, идей, гипотез, наблюдений и других материалов, к которым предполагается вернуться позднее.

## Структура

- `specification.md` — спецификация формата Work Item.
- `chatgpt_prompt.md` — промпт, реализующий данную спецификацию для ChatGPT.

## Принцип

Спецификация считается первичной.

Промпты рассматриваются как одна из возможных реализаций спецификации для конкретной модели ИИ.

При изменении спецификации промпты могут быть обновлены без изменения самих Work Items.
